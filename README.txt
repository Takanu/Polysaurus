Installation Instructions
- Compress the Polysaurus folder into a ZIP file
- In Blender, go to System Preferences > Add-ons
- Click on the Install button at the bottom of the window
- Select the Polysaurus ZIP file using the newly opened file browser
- Click on the tickbox next to the addon name in the add-ons list to activate it.

Tadaa!
